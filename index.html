Proyectos

Sistema de Gestión de Parques Naturales

Diseñé y desarrollé una base de datos en MySQL para gestionar parques naturales en Colombia, incluyendo departamentos, áreas, especies, personal, proyectos de investigación, visitantes y alojamientos. La base está normalizada hasta la 4ª Forma Normal y contiene datos realistas para 20 registros por tabla, con un diseño escalable a 50. Utilicé SQL para crear tablas, insertar datos y asegurar relaciones coherentes.

Tecnologías: MySQL, SQL

Características: Normalización, claves foráneas, datos semi-realistas.
